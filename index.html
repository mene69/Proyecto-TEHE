Nuestros servicios

Contamos con tres servicios en nuestro proyecto :

Aplicación Web (Donde usted se encuentra ahora)

Nuestra aplicación web consta de un servicio de llamadas, donde el cliente grabará (Audio y video), para que este sea mostrado en nuestro producto núcleo; la caja holográfica.

Tutorial de la aplicación web

Aplicación Móvil

La aplicación móvil integra el servicio de llamadas al igual que la aplicación web, con el fin de generar grabaciones y generar el soporte para que se puedan colocar en la caja holográfica.

Caja Holográfica

La Caja holográfica es el núcleo del proyecto; En la cual será una caja hecha con materiales reciclables, de forma rectangular (4*4) con cristales en todas sus caras. ésta tiene un cristal en su interior en 45° grados. Con ello se refleja la luz de un dispositivo Móvil (Idealmente se utiliza una Tablet) generando un efecto holográfico.

Sobre nuestro sitio

¿Que me puede ofrecer esta pagina web a comparación de otras?

En esta página web al contrario de otras, se podran ser grabados por video y audio para poder ser retransmitidos holograficamente en vivo en un proyector especializado.

¿En que puede beneficiarme este servicio?

El principal beneficio de este producto es generar un apoyo entre un emisor y un receptor. Ej: Un profesor con un alumno en una videollamada mas elaborada y mas cercana

¿Cuánto dinero necesito para tener este producto?

Actual mente nuestro producto no tiene fines de lucro, pero sí los planes para ganar ingresos es poder asociarnos a diversas intituciones para ofrecer nuestro servicios

¿Porque decidieron hacer este proyecto?

Nosotros decidimos hacer este proyecto para apoyar y benificiar a las personas con escazo acceso a clases precenciales en sectores rurales alejados de ciertos institutos educacionales.
